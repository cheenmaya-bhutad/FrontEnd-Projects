Meta tags are used to describe something about the webpage

This info is not displayed on the web page

Title - Allows us to name the webpage name to be displayed in the browser tab. This should be about your web page.

meta tag description gives a small summary about the web page. This should also highlight the key things about the webpage

meta tag keywords are provided to indicate some keywords that the page contains

meta tag robots does not allow the page to be indexed by search engine provided in the title and description. with contents the page is not indexed even for info in the body.

The title and description tags should be properly framed as the search engine searched this info to be rank the pages in seach.

To prove the above point go to any site search on google and check the web pages. The title and description of each web page is defined as search results.
